AI-Powered Invoice Generator
The smartest way to bill. This full-stack MERN application leverages the Google Gemini AI to automate the tedious parts of freelancing: creating invoices from scratch and chasing payments.

The Problem & AI Solution
Most invoicing apps require manual data entry for every line item. This project solves that by allowing users to:

Prompt-to-Invoice: Paste a rough summary of work (e.g., "3 hours of React dev at $50/hr and a $200 flat fee for SEO") and let Gemini structure it into a formal invoice.

Intelligent Summaries: The AI analyzes your billing history to provide a "Business Health" summary on your dashboard.

Built With
Frontend: React.js, Tailwind CSS (Modern, Responsive UI)

Backend: Node.js, Express.js

Database: MongoDB & Mongoose

AI Integration: Google Gemini Pro API

Authentication: JWT (JSON Web Tokens)

PDF Logic: Client-side PDF generation for instant downloads.

Features
AI Data Extraction: Uses LLMs to parse unstructured text into JSON invoice objects.

Secure Dashboard: User-specific data isolation and secure login.

Invoice Management: Full CRUD operations (Create, Read, Update, Delete) for your records.

Dynamic PDF Styling: Professional templates that look great for clients.

Financial Insights: Real-time calculation of total revenue and pending amounts.
